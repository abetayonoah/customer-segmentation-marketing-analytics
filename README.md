# Customer Segmentation using K-Means (R)

## Project Overview
This project was completed as part of the Marketing Analytics course at the University of Niagara Falls.

The objective of the analysis was to identify distinct customer segments based on financial behavior using clustering techniques. Customer segmentation helps organizations better understand their customer base and develop targeted marketing strategies.

## Tools & Technologies
- R Programming
- K-Means Clustering
- Data preprocessing and scaling
- Data visualization

## Dataset
The dataset contains customer financial and demographic attributes such as:

- Age
- Income level
- Personal savings
- Credit card usage
- Spending behavior

These variables were used to group customers into meaningful segments.

## Methodology
The following steps were performed:

1. Data cleaning and preprocessing
2. Standardization of variables
3. Determination of optimal number of clusters
4. Implementation of K-Means clustering
5. Visualization and interpretation of clusters

## Key Insights
The analysis identified multiple distinct customer groups including:

- Financially constrained customers
- Moderate income and moderate spending customers
- High income customers with strong saving behavior

These insights can help companies design targeted marketing campaigns for each segment.
